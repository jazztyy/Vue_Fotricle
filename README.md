# Fotricle 線上訂餐平台
----
> 使用 Vue.js 搭配 Tailwind 打造的線上訂餐平台。消費者可於線上訂餐，並查看餐車營業狀態及地點，使用 Google Map API，方便搜尋周遭餐車。餐車可用於上架商品並使用 POS 系統紀錄訂單資訊及數據統計。

## Demo
[Demo](http://fotricle.rocket-coding.com/index.html#/Home)

## 平台功能
* 會員註冊 / 登入
* 查看及篩選餐車 (可於 google 地圖顯示並查看路線)
* 個人資訊編輯
* 上架商品
* 購買商品

## 使用技術
* Vue
* Vue CLI
* Vue Router
* Tailwind
* Git
* Google Map API
